### Консольный калькультор
---
Поддерживаемые операции:
* **"()"** - скобки
* **"\+"**  - сложение
* **"\-"**  - вычитание
* **"\*"**  - умножение
* **"/"**  - деление
* **"^"**  - степень

---
make.cmd - компиляция. Файл =.exe скопировать в папку из PATH, например C:\WINDOWS

---
Пример использования: = 8.7 - 6 * (5.4 - 3)^2
